ganeti-cleaner

Name

ganeti-cleaner -- ganeti job queue cleaner

Synopsis

ganeti-cleaner

DESCRIPTION

The ganeti-cleaner is a periodically run script to clean old job files from the job queue archive.

ganeti-cleaner automatically removes all files older than 21 days from /usr/local/var/lib/ganeti/queue/archive.
