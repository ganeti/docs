class GlusterStorage(base.BlockDev):

View In Hierarchy

File device using the Gluster backend.

This class represents a file storage backend device stored on Gluster. Ganeti mounts and unmounts the Gluster devices automatically.

The unique_id for the file device is a (file_driver, file_path) tuple.
