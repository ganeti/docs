def GetVersion(self): ¶

Return the DRBD version.

This will return a dict with keys:

  • k_major
  • k_minor
  • k_point
  • k_fix (only on some drbd versions)
  • k_fix_separator (only when k_fix is present)
  • api
  • proto
  • proto2 (only on drbd > 8.2.X)
